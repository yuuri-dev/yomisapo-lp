プライバシーポリシー

よみサポ（iOS）

最終更新日：2026年4月17日

はじめに

Yuuri Inoue（以下「開発者」）は、アプリ「よみサポ」（以下「本アプリ」）における ユーザーのプライバシーを尊重し、個人情報の保護に取り組んでいます。 本ポリシーでは、本アプリが取り扱うデータの内容・目的・範囲について説明します。

収集・送信するデータ

本アプリは、特定の機能の利用時にのみ以下のデータを外部サービスへ送信します。

データの種類 送信先 用途
書籍の検索クエリ（ISBN・書名） Google Books API 本の検索・追加機能
読書メモの内容 AI要約サービス AIまとめ機能（使用時のみ）
用語名 Wikipedia API 用語の意味検索（使用時のみ）
いずれの送信も、ユーザーが該当機能を明示的に使用した場合にのみ行われます。 開発者がこれらのデータを収集・保存・販売することはありません。

カメラの使用について

本アプリはバーコードスキャン機能のためにカメラへのアクセスを求める場合があります。 カメラ映像は端末内でのバーコード解析にのみ使用し、外部へ送信することは一切ありません。

収集しないデータ

本アプリは以下のデータを収集しません。

  • 氏名・メールアドレス・電話番号などの個人識別情報
  • 位置情報（GPS）
  • 広告ID（IDFA / GAID）
  • 閲覧履歴・行動ログ

データの保存先

本棚データ・読書メモ・用語などのユーザーデータは、すべてお使いの端末内（SQLite）にのみ保存されます。 開発者の外部サーバーへの保存、第三者への提供・販売は行いません。

第三者サービスのプライバシーポリシー

本アプリが利用する外部サービスには、それぞれのプライバシーポリシーが適用されます。

  • Google Books API：https://policies.google.com/privacy
  • Wikipedia API：https://foundation.wikimedia.org/wiki/Privacy_policy
